How to Run the Code for Transformer Blocks

This document provides instructions on how to run the code for the Transformer Blocks assignment. Follow the steps below to ensure everything is set up correctly.

Part 1: Encoder

To run your implementation of the Encoder, execute the following command in your terminal:

python main1.py

Part 2: Decoder

To run your implementation of the Decoder, execute the following command in your terminal:

python main2.py

Part 3: Architectural Exploration

To run your implementation of the ALiBi transformer, execute the following command in your terminal:

python main3.py

Note
Make sure you are running these commands in the terminal from the directory where your assignment files are located. Each script corresponds to different parts of the assignment and should be run in the order outlined above to ensure a smooth progression through the tasks.
